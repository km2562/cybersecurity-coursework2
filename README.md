# Project Title

HAR Analyser

## Description

This is a bespoke C# console app written to convert a specific HAR file from JSON and to extract key data items to csv
It has a second purpose to search for strings within the given webpages.
This app relies on the HAR and webfiles being added to a folder on the C drive named "HAR"
The HAR file is hardcoded as "www.tkmaxx.com.har"
The webpage is hardcoded as "Payment powered by Worldpay - TK Maxx UK_files"

## Executing program

The app was installed locally and run with Visual Studio
